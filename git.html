The Diagnosis of the above scenario

When we write:

git remote add origin [url]

It means that we are adding a remote repository of [url] and naming it as "origin".

git remote add origin https://github.com/profile/rep.git

Do not forget that github remote repo have .git extension in end of repository name.

git remote -v

This command displays all the messed up remote repositories you have made

git remote remove origin

This command deletes/ removes the messed up remote repository you made.
